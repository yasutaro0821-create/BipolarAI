# 今すぐやること（簡単手順）

## ステップ1: コードをGASエディタに貼り付ける

### 1. Code.gsを開く

1. エクスプローラーで `C:\Users\yasut\BipolarAI` を開く
2. `Code.gs` をダブルクリック（メモ帳で開く）

### 2. コードをコピー

1. `Ctrl+A`（全選択）
2. `Ctrl+C`（コピー）

### 3. GASエディタを開く

ブラウザで以下を開く：

```
https://script.google.com/home/projects/1CQCYu6NIh_6AODKesZQL3Ng6nz4BtFZU5lyEmGP84a7f2EYpkt4dSk0S/edit
```

### 4. コードを貼り付け

1. GASエディタのコードエリアで `Ctrl+A`（全選択）
2. `Delete`（削除）
3. `Ctrl+V`（貼り付け）

### 5. 保存

`Ctrl+S` または「保存」ボタン（💾）

---

## ステップ2: OpenAI APIキーを設定

### 1. プロジェクトの設定を開く

GASエディタで「プロジェクトの設定」（⚙️）をクリック

### 2. スクリプトプロパティを追加

1. 「スクリプト プロパティ」セクションで「スクリプト プロパティを追加」をクリック
2. 以下を設定：
   - **プロパティ**: `OPENAI_API_KEY`
   - **値**: 
     ```
     YOUR_OPENAI_API_KEY（config.jsonを参照）
     ```
3. 「保存」をクリック

---

## ステップ3: デプロイを更新

### 1. デプロイを管理

GASエディタで「デプロイ」→「デプロイを管理」をクリック

### 2. 既存のデプロイを編集

1. 既存のデプロイの右側の「編集」（✏️）をクリック
2. 「バージョン」を「新バージョン」に変更
3. 「デプロイ」をクリック

**重要**: デプロイを更新しないと、コードの変更が反映されません。
