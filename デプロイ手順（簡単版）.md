# 双極AI GAS デプロイ手順（簡単版）

## 毎回やること（コピペでOK）

以下のコマンドをコピーしてPowerShellで実行するだけです：

```powershell
cd C:\Users\yasut\BipolarAI_clasp; .\deploy.ps1
```

これで完了です！既存のURLが自動的に維持されます。

## 何が起こるか

1. コードをGASにアップロード
2. 既存のデプロイを更新（URLは変わらない）
3. 完了！

## 初回のみ

初回実行時は、既存のデプロイIDを自動的に取得して保存します。2回目以降は、保存されたデプロイIDを自動的に使用するので、何も考える必要はありません。

## トラブルシューティング

### エラーが出た場合

`.deployment-id`ファイルを削除して、再度実行してください：

```powershell
cd C:\Users\yasut\BipolarAI_clasp
Remove-Item .deployment-id -ErrorAction SilentlyContinue
.\deploy.ps1
```

---

**重要**: このコマンドを実行するだけで、既存のURLが維持されます。新しいURLが生成されることはありません。
