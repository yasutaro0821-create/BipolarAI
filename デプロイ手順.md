# 双極AI GAS デプロイ手順

## 毎回コピペ用コマンド

### 方法1: 新しいデプロイを作成（簡単）

```powershell
cd C:\Users\yasut\BipolarAI_clasp; clasp push; clasp deploy --description "Update $(Get-Date -Format 'yyyy-MM-dd HH:mm:ss')"
```

### 方法2: スクリプトを使用（推奨）

```powershell
cd C:\Users\yasut\BipolarAI_clasp
.\deploy.ps1
```

## 詳細説明

### clasp push + deploy の流れ

1. **`clasp push`**: コードをGASエディタにアップロード
2. **`clasp deploy`**: ウェブアプリとしてデプロイ（新しいURLを生成）

### 既存のデプロイを更新する場合

既存のデプロイIDを確認：
```powershell
cd C:\Users\yasut\BipolarAI_clasp
clasp deployments
```

デプロイIDをコピーして、以下のコマンドで更新：
```powershell
cd C:\Users\yasut\BipolarAI_clasp
clasp deploy --deploymentId YOUR_DEPLOYMENT_ID --description "Update $(Get-Date -Format 'yyyy-MM-dd HH:mm:ss')"
```

## 注意事項

- 新しいデプロイを作成すると、**新しいURLが生成されます**
- 既存のデプロイを更新すると、**URLは変わりません**（推奨）
- 初回のみ、GASエディタでデプロイ設定（実行ユーザー、権限など）を確認してください
